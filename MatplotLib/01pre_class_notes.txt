ðŸ§  What is Matplotlib?
Matplotlib is a Python library used to create charts and graphs. It helps visualize data to find patterns, trends, and insights.

ðŸ”§ Basic Structure

import matplotlib.pyplot as plt

x = [1, 2, 3, 4]
y = [10, 20, 25, 30]

plt.plot(x, y)        # Plot line chart
plt.show()            # Show the chart

ðŸ“Š Common Types of Plots
Plot Type	Function Used	Use Case
Line Plot	plt.plot()	Trend over time
Bar Chart	plt.bar()	Compare categories
Histogram	plt.hist()	Distribution of data
Scatter Plot	plt.scatter()	Relationship between two variables
Pie Chart	plt.pie()	Proportions (percentage share)
ðŸŽ¨ Customization Options

plt.title("My Chart")               # Add title
plt.xlabel("X-Axis")                # Label X
plt.ylabel("Y-Axis")                # Label Y
plt.grid(True)                      # Add grid
plt.legend(["Sales"])              # Add legend
plt.xlim(0, 10)                     # X-axis limit
plt.ylim(0, 100)                    # Y-axis limit

ðŸŒˆ Styling Your Plot

plt.plot(x, y, color='green', linestyle='--', marker='o')

color: red, blue, green, etc.
linestyle: '-', '--', ':'
marker: 'o', '*', 's', etc.
ðŸ“Œ Subplots (Multiple Charts Together)

plt.subplot(1, 2, 1)  # (rows, cols, chart_number)
plt.plot(x, y)

plt.subplot(1, 2, 2)
plt.bar(x, y)

plt.show()
